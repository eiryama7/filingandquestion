üîß FOUNDATION MODELS - HOW IT WORKS
================================================================================

The core integration is in Services/LLMChatService.swift:

1. CHECK AVAILABILITY
   SystemLanguageModel.isSupported
   SystemLanguageModel.default.availability

2. CREATE SESSION
   let model = SystemLanguageModel.default
   let session = try model.makeSession()

3. SEND MESSAGE & GET RESPONSE
   let response = try await session.respond(to: userMessage)

The session automatically maintains conversation history, enabling
multi-turn dialogues with context awareness.
